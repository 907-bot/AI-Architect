## One-time UE5 setup (do this once)

### 1. Enable required plugins
Open Unreal Engine 5 → Edit → Plugins → search and enable:
- ✅ Python Editor Script Plugin
- ✅ Interchange Framework
- ✅ glTF Exporter
- ✅ Datasmith Importer
- ✅ Sun Position Calculator

### 2. Enable Lumen + Nanite
Project Settings → Rendering:
- ✅ Global Illumination: Lumen
- ✅ Reflections: Lumen  
- ✅ Nanite: enabled
- ✅ Ray Tracing: enabled (optional, for path tracing)
- ✅ Sky Atmosphere: enabled

### 3. Set Python path (first time only)
Edit → Project Settings → Python → Additional Paths
→ Add the folder containing this README

## Run the import (every time you export a new building)

### Option A — Quick (recommended)
1. Open UE5 with your project
2. Tools → Execute Python Script
3. Select `ue5_setup.py` from this folder
4. Wait ~30 seconds → building appears in viewport ✓

### Option B — Manual drag-and-drop
1. Drag `building.glb` into UE5 Content Browser
2. Accept import defaults (meshes + materials)
3. Drag imported mesh into viewport
4. Run lighting setup manually (see below)

## Walkthrough setup (First Person)
1. Add → Blueprint Class → Character → name it `BP_ArchWalk`
2. Set Default Pawn Class in World Settings → `BP_ArchWalk`
3. Press Play → WASD to walk, mouse to look
4. Or use Unreal's built-in **Walkthrough** camera mode:
   Viewport → Perspective → Cinematic Viewport → Fly mode (RMB + WASD)

## Drone flythrough (Sequencer)
1. Window → Cinematics → Sequencer → New Sequence
2. Add Track → Camera Cut → add your `ArchCamera`
3. Add keyframes along a path around/through the building
4. Render → Movie Render Queue → Path Tracer preset
5. Output: 4K ProRes or H.265

## Path Tracing (ultra-photorealistic still renders)
In UE5 Python console:
```python
import unreal
# Enable path tracer
unreal.SystemLibrary.execute_console_command(None, "r.PathTracing 1")
unreal.SystemLibrary.execute_console_command(None, "r.PathTracing.MaxBounces 16")
unreal.SystemLibrary.execute_console_command(None, "r.PathTracing.SamplesPerPixel 4096")
```
Then: Window → Movie Render Queue → add level sequence → Output: EXR → Render

## Recommended UE5 scalability settings for architecture
```
r.Shadow.Virtual.Enable 1          # Virtual Shadow Maps
r.Lumen.Reflections.ScreenTraces 1
r.Lumen.TranslucencyReflections.FrontLayer.Enable 1
r.RayTracing.GlobalIllumination 0  # Use Lumen instead
r.RayTracing.Reflections 1         # Ray-traced glass reflections
sg.PostProcessQuality 4
sg.ShadowQuality 4
```
